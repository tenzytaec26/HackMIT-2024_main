## Getting Started

### Prerequisites

- Node.js (for backend)
- React.js (for frontend)

### Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/ThomasC86/HackMIT-2024
   cd HackMIT-2024
   ```

2. **Set up the backend**:

   ```bash
   cd backend
   npm install
   npm start
   ```

3. **Set up the frontend**:
   ```bash
   cd frontend
   npm install
   npm start
   ```

## Start Your Skin Assessment
